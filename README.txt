BetterBox'd Movie Software

    BetterBox'd is a database-driven movie platform built with Python and SQL that enables users to watch, rate, 
    and organize movies, follow other users, and create personalized playlists.

    I helped design an extensive EER/relational schema modeling users, movies, contributors 
    (actors/directors/publishers), genres, studios, platforms, and social interactions, then 
    implemented the schema in PostgreSQL with data validation, constraints, and realistic test data generated using Python’s Faker library.

    We also built advanced SQL analytics and a recommendation system that suggests movies 
    based on users’ highly rated genres and viewing history, demonstrating skills in data modeling, 
    querying, and data-driven feature design.


HOW TO RUN IT:
